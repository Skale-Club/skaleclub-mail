Create `src/server/lib/pagination.ts` with the following exports:

1. **`paginationQuerySchema`** — Zod schema for pagination query params:
   ```typescript
   export const paginationQuerySchema = z.object({
       page: z.coerce.number().int().min(1).default(1),
       limit: z.coerce.number().int().min(1).max(100).default(25),
   })
   ```

2. **`PaginationMeta`** type:
   ```typescript
   export interface PaginationMeta {
       page: number
       limit: number
       total: number
       totalPages: number
   }
   ```

3. **`PaginatedResult<T>`** type:
   ```typescript
   export interface PaginatedResult<T> {
       data: T[]
       pagination: PaginationMeta
   }
   ```

4. **`paginate()`** function — generic function that performs count query + data query:
   ```typescript
   export async function paginate<T>(
       db: any,
       table: any,
       options: {
           where?: any
           page: number
           limit: number
           orderBy?: any
           with?: Record<string, any>
           columns?: Record<string, boolean>
       }
   ): Promise<PaginatedResult<T>>
   ```

   Implementation:
   - Calculate `offset = (page - 1) * limit`
   - Count query: `db.select({ count: sql\`count(*)\` }).from(table).where(options.where)`
   - Data query: `db.query[tableName].findMany({ where, limit, offset, orderBy, with, columns })`
   - Return `{ data, pagination: { page, limit, total, totalPages: Math.ceil(total / limit) } }`

   Use Drizzle's `getTableName(table)` to get the query key for `db.query[tableName]`.
   Import `sql` and `getTableName` from `drizzle-orm`.
  </action>
